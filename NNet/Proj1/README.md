# 运行方式 

+ `requirements: python==3.8.5, numpy==1.19.1, matplotlib==3.3.1`

+ 可选参数：

  ```shell
  --iter_nums           迭代次数
  --seed                随机种子
  --lr                  学习率
  --hidden_dim          隐藏层维度
  --hidden_layer_num    隐藏层数目
  --input_dim           输入层维度
  --output_dim          输出层维度
  --activation          激活函数类型
  --data_nums           数据大小
  ```

  命令行键入

  ```shell
  python main.py
  ```

  即可使用默认参数运行。
